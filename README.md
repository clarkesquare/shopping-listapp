# 🛒 Shopping List App

간단하고 우아한 쇼핑 리스트 앱입니다.

## 기능

- ✅ 아이템 추가/삭제/체크
- 🌙 라이트/다크 모드 지원
- 💾 localStorage를 통한 데이터 저장
- 📱 반응형 디자인 (모바일/태블릿/데스크톱)
- 🔒 XSS 공격 방어
- 🔄 다중 탭 동기화

## 사용 방법

1. `shopping-list.html` 파일을 브라우저에서 열기
2. 아이템 입력 후 "추가" 버튼 클릭 또는 Enter 키
3. 체크박스 클릭으로 완료 표시
4. 호버 후 × 버튼으로 삭제
5. 우측 상단 버튼으로 테마 변경

## 테스트 실행

```bash
npm install
npm test
```

## 기술 스택

- HTML5
- CSS3 (CSS Custom Properties)
- Vanilla JavaScript
- Playwright (테스트)

## 스크린샷

라이트 모드와 다크 모드를 지원합니다.
